# 知乎图片回答筛选器

一个提供只看知乎含图片内容的浏览器扩展，帮助用户过滤低质量内容，专注于优质回答。

## 功能特点

- 🎯 智能过滤无图内容
- 🎨 优化阅读体验
- ⚡ 轻量级，不影响浏览性能

## 安装方法

1. 克隆仓库
```bash
git clone https://github.com/yourusername/high-quality-zhihu.git
cd high-quality-zhihu
```

2. 在Chrome浏览器中加载扩展
- 打开Chrome，进入`chrome://extensions/`
- 开启右上角的"开发者模式"
- 点击"加载已解压的扩展程序"
- 选择项目目录

## 使用说明

1. 安装完成后，在知乎页面会自动启用扩展
2. 点击浏览器工具栏的扩展图标可以：
   - 开启/关闭过滤功能
   - 调整过滤规则
   - 自定义显示设置

## 开发说明

本项目使用以下技术栈：
- JavaScript (ES6+)
- Chrome Extension API
- CSS3
